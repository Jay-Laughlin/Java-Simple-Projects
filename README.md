# Java-Simple-Projects

These examples are meant to explore different programming concepts like data structures and algorithms as well as different applications like physics, rendering and graphics. They are often coded in 1 file (to make them easier to upload/download for LMS's) and will prioritize the element that I'm exploring and so may not always have polished code elsewhere (hardcoded constraints, access levels and other OOPs violations). 

CarRotation - is a simple attempt to explore car physics for turning and accelerating. I'm trying to explore turning (rotation using AffineTransform) and drifting physics in an extensible swing application.
